\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_shader}{Shader}}. 

Creates, compiles and links a shader from the file location passed~\newline
The passing of a file location gives the shader more flexibity so there is no hardcoded shader in this object~\newline
\mbox{\hyperlink{class_shader_a8cd68191ca7f7430ed37384d1d78b5a8}{create\+Shader()}} will destroy the current program held by this object and create a new one so becareful~\newline
Program will terminate if there\textquotesingle{}s any errors
