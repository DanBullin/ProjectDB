\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_model}{Model}}. 

By\+: Dan Bullin (P14132258)~\newline
This class creates a Open\+GL V\+AO, multiple GL V\+B\+Os to store data for a model~\newline
This class is an OO implementation of a V\+AO~\newline
While the model class can be instantiated, templates have been made such as Unfolded \mbox{\hyperlink{class_cube}{Cube}}, \mbox{\hyperlink{class_kite}{Kite}}, etc which inherit this class. Instantiate them~\newline
Child classes should call \mbox{\hyperlink{class_model_a3e389a8872837f0b1ac4c3e14d9b42cb}{create\+Model()}} in their constructor Call \mbox{\hyperlink{class_model_a105aeca6a0c7389edf931eff6a91b1ab}{bind()}} to bind this object\textquotesingle{}s V\+AO
