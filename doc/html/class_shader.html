Detailed Description

A Shader.

Creates, compiles and links a shader from the file location passed
The passing of a file location gives the shader more flexibity so there is no hardcoded shader in this object
createShader() will destroy the current program held by this object and create a new one so becareful
Program will terminate if there's any errors
