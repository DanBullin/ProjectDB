Detailed Description

A Model.

By: Dan Bullin (P14132258)
This class creates a OpenGL VAO, multiple GL VBOs to store data for a model
This class is an OO implementation of a VAO
While the model class can be instantiated, templates have been made such as Unfolded Cube, Kite, etc which inherit this class. Instantiate them
Child classes should call createModel() in their constructor Call bind() to bind this object's VAO
